About Digital Registers

A register is a small amount of fast storage in a computer's CPU. This simulation shows an 8-bit register that can store values from 0 to 255.

Key operations:

  • Load: Stores a new value in the register
  • Shift Left/Right: Moves bits left or right (multiplies/divides by 2)
  • Increment/Decrement: Increases/decreases the value by 1
  • Clear: Resets all bits to 0
